How to switch to real Binance
1. Register testnet.binance.vision for API keys

2. Set environment variables and restart:
   export SIMULATE_MODE=0
   export BINAKE_API_KEY="your_key"
   export BINAKE_API_SECRET="your_secret"

3. Enable WeChat notifications:
   a) Visit pushplus.plus — scan QR with WeChat
   b) Copy Token and set:
   export PUSHPLUS_TOKEN="your_token"
   export PUSH_ENABLED=1

WARNING: Always set IP whitelist before live trading
